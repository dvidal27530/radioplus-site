Support – RadioPlus

À propos de RadioPlus

RadioPlus est une application iOS qui permet d’écouter facilement une mosaïque de radios en streaming, classées par genre et type de contenu. L’application s’appuie sur un backend sécurisé pour récupérer les flux et les informations éditoriales (logos, noms, catégories).

Contacter le support

Si vous rencontrez un problème avec l’application ou si vous souhaitez nous envoyer une suggestion, vous pouvez nous contacter à l’adresse suivante :

ton-email-support@example.com

Nous faisons notre possible pour répondre dans les meilleurs délais.

Signaler un bug ou un problème de lecture

Pour nous aider à corriger rapidement un problème, merci de préciser les informations suivantes dans votre message :

  • Le modèle de votre iPhone (par exemple iPhone 13, iPhone SE, etc.).
  • La version d’iOS installée.
  • Le nom de la radio concernée, si le problème ne touche qu’un flux.
  • Une description la plus précise possible du problème rencontré.

Questions et suggestions

Vous pouvez également utiliser cette adresse pour nous envoyer vos idées d’améliorations, proposer de nouvelles radios à ajouter ou nous faire part de tout retour sur l’expérience utilisateur.
